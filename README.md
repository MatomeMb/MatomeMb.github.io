# Matome Mb — Portfolio

This repository contains a single-page professional portfolio for Matome Mb, built with plain HTML and CDN-only assets (Tailwind CSS, Font Awesome, AOS). The site is intended to be published via GitHub Pages.

Live site
- Primary (repo-based): https://matomembowene.github.io

Features
- Responsive, dark themed single-page portfolio
- Hero, About, Production Projects (3), Academic Projects (5), Skills, Contact (Formspree placeholder), Footer
- Smooth scroll, AOS animations, hover effects
- No build tools — uses CDN assets only

Setup / Local preview
1. Clone the repo or ensure you are in the repo directory:

```cmd
cd C:\path\to\matomembowene.github.io
```

2. Open `index.html` directly in a browser for a local preview, or run a simple static server. Example using Python 3 (optional):

```cmd
python -m http.server 8000
# then open http://localhost:8000
```

Form handling
- The contact form action is a Formspree placeholder: `https://formspree.io/f/yourFormId`. Replace `yourFormId` with your Formspree form ID (or another endpoint) to receive messages.

Publishing
- GitHub Pages: ensure Pages is enabled in the repository Settings → Pages, branch `main`, folder `/(root)`.
